Templates

Ca n'est pas compliqué :

  • tu fais un fichier html template de base (par ex base.html) que tu mets dans le répertoire templates de ton app ou dans un repertoire global definit dans le parametre DIRS de TEMPLATES
  • tu y inclus des blocs du style

    {% block mon_bloc %}

    du html...

    {% endblock %}

  • tu inclus dans tes templates un {% extends 'base.html' %}
  • tu configures une route dans le fichier urls.py de ton app qui pointe sur une vue
  • dans cette vue, tu fais appel à la fonction render() en lui passant un template en parametre...
En plus de cela, tu peux utiliser des balises pour faire :
  • l'inclusion de variables avec {{ ma_variable }}
  • des boucles avec {% for xx in yy %}
  • des conditions avec {% if mon_parametre == 2 %}
  • de l'inclusion d'un autre template avec {% include 'autre/template.html' %}

Django c'est drôlement chouettte !
